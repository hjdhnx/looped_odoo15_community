Employee Monthly Gratuity Accrual Posting

Employee Monthly Gratuity Accrual App for Odoo.

This module will create monthly employee gratuity sheet and create draft accounting entry at the end of every month.
This app will work with Odoo community and Enterprise version. For more details you can see below screenshots.

Menus Available:

  • Payroll / Gratuity / Gratuity Sheet
  • Payroll / Gratuity / Create Gratuity Sheet

Main Features:

  • Allow you to create monthly gratuity accrual accounting entries.
  • Gratuity rule : 21 days basic for 1-5 years and 30 days basic for above five years of service.
  • If employee terminated after one year, no entitlement of gratuity if the service is less than 1 year.
  • 1-3 Years : 1/3rd of Gratuity (7 Days)
  • 4-5 Years : 2/3rd of Gratuity (14 days)
  • >5 Years : 30 days basic for above five years of service
  • Employee wise gratuity sheet for each month created.
  • Module contain excel sheet for employee gratuity calculation in detail.
  • Created gratuity amount of each employee as provision every month.
  • For more details see below screenshots. You can contact us if you have any query.
  • Module will work with odoo version 13.0 community and enterprise.
